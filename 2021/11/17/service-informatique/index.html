Vous tombez en plein dans l’un des piliers de l’internet.

La résolution de noms .

Votre site web a un nom , c’est le nom de domaine .(point)fr, ou .(point autre chose)

nom ecriture
racine . <- c’est la racine : la racine est un point.
Top Level
Domain http://example.com http://example.org .edu .gouv.fr .jp .cn .fr
Domain Name https://ordinatous.com (point)
sub.domain https://carto.ordinatous.com
arborescence https://ordinatous.com/flex/

C’est simple en fait .

Vous payez un registrar , qui enregistre votre nom de domaine.TLD.

  • TLD : pour Top Level Domain .com .org .edu
    • ou cc : pour country code, par exemple .fr
  • domain name : exemple1
  • sous domaine : comme : carto.ordinatous tous les enregistrements , et redirecteurs. (Et pourquoi ces bases de données ?)
